# BazarSetu - Hyperlocal Marketplace Startup

> **Local dukaan se direct ghar tak delivery.**

BazarSetu is a modern hyperlocal marketplace platform designed to bridge the gap between neighborhood vendors and local customers. By empowering small businesses with digital tools and providing users with instant access to nearby services and products, we are redefining the local commerce experience.

## 🚀 Key Features

- **🏠 Hyperlocal Delivery**: Get products from your favorite nearby grocery, staples, fruits, and vegetables stores delivered in minutes.
- **🛠️ Service Booking**: On-demand access to verified local service providers including plumbers, electricians, and laundry services.
- **📦 Real-time Tracking**: Monitor your orders and service providers in real-time for ultimate reliability.
- **📈 Vendor Dashboard**: empowering local sellers with powerful digital tools to manage inventory and scale their reach.
- **⚡ Fast & Affordable**: Optimized logistics to ensure rapid delivery at competitive local prices.

## 💻 Technology Stack

This portfolio website is built with a focus on modern aesthetics and high performance:

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Glassmorphism UI)
- **Logic**: Vanilla JavaScript
- **Animations**: GSAP & ScrollReveal for smooth scroll interactions
- **Icons**: Font Awesome 6 (including the latest branding like the X logo)
- **Responsive**: Fully optimized for Desktop, Tablet, and Mobile devices

## 📂 Project Structure

- `index.html`: Main landing page featuring Hero, Features, How It Works, Market Analysis, and Team.
- `css/style.css`: Core design system using CSS variables, flexbox, and grid layouts.
- `Script/script.js`: Interactive elements, scroll animations, and navigation logic.
- `html/`: Legal documentation subpages (Privacy, Terms, Refund, Security).
- `images/`: High-quality assets including app mockups and professional team portraits.
